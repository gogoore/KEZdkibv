# 项目简介

基于深度学习的YOLOv11花生种子霉变检测系统，运用先进的目标检测技术对花生种子进行实时监测。系统包括YOLO数据集处理、UI界面展示、登录注册模块，以及项目完整源码。技术特点在于采用YOLOv11算法，提高检测速度与准确性。该系统主要用于农业领域，有助于提升花生种子质量检测的自动化水平。


## 核心功能
- 基于YOLOv11算法实现花生种子霉变实时检测
- 提供YOLO数据集处理功能
- 包含用户登录注册界面
- 提供系统UI界面展示检测结果
- 提供完整项目源码

## 适用场景
- 农业领域花生种子质量检测
- 种子霉变自动化监测
- 种子质量评估与分析

## 技术特点
- 采用YOLOv11目标检测算法
- 提高检测速度与准确性
- 支持实时监测功能
- 完整的深度学习项目实现
